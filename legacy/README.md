
##Cloud Native Infographic




Install components:

```
npm install
```

To run Dev:
```
gulp dev
```


To only build:
```
gulp build
```


The final build will be located in the `dist` folder

```
|- dist/
      |- css/
      |- fonts/
      |- images/ 
      |- js/
      |- index.html
  
```


In case you dont have NPM or GULP installed:

```
//To install NPM:
http://blog.npmjs.org/post/85484771375/how-to-install-npm

//To install Gulp CLI:
npm install gulp-cli -g
```
